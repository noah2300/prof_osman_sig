# Prof. Osman Email Signature

Static site (Vercel-ready) to preview and copy **Prof. Osman M. Bakr**'s KAUST email signature.

## Files

- `index.html` — preview + copy button
- `signature.html` — the email signature markup
- `signature.txt` — plain-text fallback
- `kaust-logo.png` — KAUST logo used in the signature
- `vercel.json` — minimal Vercel config

## Deploy on Vercel

1. Push this repo to GitHub
2. In Vercel: **New Project** → Import `noah2300/prof_osman_sig`
3. Framework preset: **Other** (static)
4. Deploy

## Use in Gmail / Outlook

Open the deployed site, click **Copy Signature**, then paste it into your email client's signature settings.
